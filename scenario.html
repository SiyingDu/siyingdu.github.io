Instructional Design | Elearning for System Training


Background

In travel industry, agents in the customer service centers need to handle different complicated systems/softwares while they serving the customers, such as Global Distribution System(GDS), Customer Management System and so on. This elearning is designed as a self-paced web based training which use simulation to make user to practise using the software during the training process


Learning Need Design Analysis & Insight

By analyzing and synthesizing the information we gathered from subject matter experts and user interviews, we were able to see four insights:

  • Insight 1: Agents are not sure when to use what functions.
  • Insight 2: Some agents are not native english speakers, so it is hard for them to learn abstract concepts
  • Insight 3: Unfriendly mobile experience; hard to use on the go.
  • Insight 4: Some agents are technically challenged, and they require simpler and more direct user interface.

    • Learning Theroy Appiled

    Cognitive Load Theory (John Sweller) suggests that learning happens best under conditions that are aligned with human cognitive architecture. From an instructional perspective, information contained in instructional material must first be processed by working memory. For schema acquisition to occur, instruction should be designed to reduce working memory load. Cognitive load theory is concerned with techniques for reducing working memory load in order to facilitate the changes in long term memory associated with schema acquisition.
